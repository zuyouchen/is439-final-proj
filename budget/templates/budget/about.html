What is EZ Budget App?

This simple web application is an online budgeting proof of concept. In theory, as a financial planning business, you'd want to manage your advisors, clients, their budgets, incomes, and expenses in a single application. Additionally, as a client, you could log in and view your budgeting information relatively easily.

The application currently just allows for superusers to perform CRUD operations on all fully supported model objects (Advisor, Client, Budget, Income, Expense, BudgetCategory). Some pre-populated object models are not editable aside from the django-admin application (ExpenseCategory, IncomeCategory, and Frequency). This is intended and due to the fact that these are realistically static, determined by the financial planning business.

Eventually, the application could transition to a more complex authorization scheme where individual users have access to their information, advisors have access to their respective clients, and administrators have access to all. This could be easily implemented given the existing code checking for permissions.

The data in this application is fictional. Some names are taken from Suits.

Navigation

Select any of the header buttons above to view lists of the respective objects.

Selecting an item from the list views sends you to a detailed page, containing related information of that particular object.

Detailed pages list object attributes alongside buttons to update or delete the object.

Budgets are broken down into separate Budget Categories, which can only be created and updated given an existing Budget.

[WIP] Selecting a filter by button from the list view sends you to a filtering page, where you can filter by category, frequency, etc.
[WIP] Selecting a "generate visualization" button from the list view sends you to a page that generates a visualization of budget/incomes/expenses
